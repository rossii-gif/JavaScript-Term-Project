Girly Games inspired by the 2000s
Developed By Isabella Rossi
CIT 190
April 2026

"Girly Games inspired by the 2000s" is exactly what the title says, a game website inspired by nostalgia growing up as a girl in the 2000s. Inspiration was heavily taken from a site I used to play on a lot called 'GirlsGoGames'. 

Game 1: Dress up Ballerina
Overview: This is a drag-and-drop dress-up game where the player can style a ballerina by selecting and moving different hair and dress images onto the doll. It uses jQuery UI’s draggable feature so each hairstyle and dress can be freely dragged around the screen. The player can also name the ballerina. There is also a slider that lets the user adjust the hue of the dresses to change the colors in real time to choose the perfect color.

Game 2: Bella’s Cooking Game 
Overview: This is an interactive cooking game where players drag and drop ingredient images into a bowl to try and create specific recipes. The game tracks which ingredients were added and checks if they match a known recipe combination. If the correct ingredients were used, the player clicks “Enter” to see what dish they have made. 

Game 3: Fairy Maze
Overview: This is a skill game where the player controls a fairy moving through an enchanted forest using the arrow keys. The goal is to avoid randomly generated tree stump obstacles that move across the screen while staying alive as long as possible. A score counter increases as the player successfully passes the stumps. The game runs on a canvas with continuously updating graphics. When the fairy hits a stump the game has ended. 

Target Audience: Obviously anyone can play it, though it was directed towards more towards women who grew up around my time in the 2000s.
